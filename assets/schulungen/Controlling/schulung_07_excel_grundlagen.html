01

Die wichtigsten Formeln im Controlling-Alltag

Excel ist das wichtigste Werkzeug im Controlling. Doch statt hunderte Funktionen auswendig zu lernen, reichen im Alltag etwa 15–20 Formeln, die man wirklich beherrscht. Hier sind die unverzichtbaren:

Grundrechenarten & Aggregation

SUMME / SUM
=SUMME(Bereich)
Addiert alle Zahlen in einem Zellbereich. Basis jeder Controlling-Auswertung.
=SUMME(B2:B13) → Jahresumsatz
SUMWENN / SUMIF
=SUMWENN(Bereich; Kriterium; Summe_Bereich)
Summiert nur Werte, die eine Bedingung erfüllen — z.B. Umsatz einer bestimmten Kostenstelle.
=SUMWENN(A:A;"KST-100";B:B)
ZÄHLENWENN / COUNTIF
=ZÄHLENWENN(Bereich; Kriterium)
Zählt Einträge nach Bedingung. Gut für Häufigkeitsanalysen oder offene Posten.
=ZÄHLENWENN(C:C;"offen")
MITTELWERT / AVERAGE
=MITTELWERT(Bereich)
Berechnet den arithmetischen Mittelwert — für Durchschnittswerte bei KPIs.
=MITTELWERT(D2:D13)
MIN / MAX
=MIN(Bereich) / =MAX(Bereich)
Findet den kleinsten bzw. größten Wert — für Ausreißer-Analyse bei Kostenpositionen.
=MAX(E2:E50) → Höchster Auftrag
ANZAHL2 / COUNTA
=ANZAHL2(Bereich)
Zählt alle nicht-leeren Zellen — für Vollständigkeitsprüfungen bei Datensätzen.
=ANZAHL2(A:A)-1 → Anzahl Datensätze

Logik & Suche

WENN / IF
=WENN(Bedingung; Dann; Sonst)
Die wichtigste Logikfunktion. Unverzichtbar für Plan-Ist-Ampeln und Abweichungsklassen.
=WENN(B2>0;"Gewinn";"Verlust")
WENNFEHLER / IFERROR
=WENNFEHLER(Formel; Wert_bei_Fehler)
Fängt Fehlerwerte (#NV, #DIV/0!) ab. Hält Reports sauber und professionell.
=WENNFEHLER(A2/B2;0)
INDEX / VERGLEICH
=INDEX(Spalte;VERGLEICH(Suchwert;Spalte;0))
Flexibler als SVERWEIS: sucht in beliebige Richtung, linke Spalten möglich.
=INDEX(D:D;VERGLEICH("KST-200";A:A;0))
RUNDEN / ROUND
=RUNDEN(Zahl; Stellen)
Rundet auf gewünschte Dezimalstellen. Wichtig für konsistente Darstellung in Reports.
=RUNDEN(A2/1000;1) → Tsd. EUR

🔢 Interaktive Demo — Plan-Ist-Berechnung

Abweichung: −15.000 € | −3,0 %
=WENN((B2-C2)/C2<−0,05;"⚠ kritisch";"✓ ok") → ✓ ok
Praxis-Tipp: Lerne zuerst SUMWENN, WENN und WENNFEHLER — diese drei Formeln lösen rund 60 % aller typischen Controlling-Aufgaben in Excel.
02

SVERWEIS & XVERWEIS — Daten verbinden

In der Praxis liegen Daten oft in verschiedenen Tabellen: Buchungen in SAP, Stammdaten in einer Planungsdatei. SVERWEIS und XVERWEIS sind die Brücke zwischen diesen Welten.

SVERWEIS — der Klassiker

Sucht einen Wert in der ersten Spalte einer Tabelle und gibt einen Wert aus einer anderen Spalte derselben Zeile zurück.

// Syntax: =SVERWEIS(Suchkriterium; Matrix; Spaltenindex; [Bereich_Verweis]) // Beispiel: Kostenstellen-Bezeichnung nachschlagen =SVERWEIS(A2; Kostenstellen!A:B; 2; 0) // A2 = gesuchte KST-Nummer // Kostenstellen!A:B = Nachschlagetabelle // 2 = Spalte 2 (Bezeichnung) // 0 = exakte Übereinstimmung (immer 0 im Controlling!)
Häufiger Fehler: SVERWEIS sucht immer in der ersten Spalte der Matrix. Liegt der Suchwert rechts vom gesuchten Wert, liefert SVERWEIS #NV. → Lösung: XVERWEIS oder INDEX/VERGLEICH nutzen.

XVERWEIS — der moderne Ersatz (Excel 365 / 2021)

XVERWEIS ist flexibler, lesbarer und fehlerresistenter als SVERWEIS. Er kann in jede Richtung suchen und hat einen integrierten Fehlerwert.

// Syntax: =XVERWEIS(Suchkriterium; Suchbereich; Rückgabe; [wenn_nicht_gefunden]; [Übereinstimmungsmodus]) // Beispiel: Kostenstellen-Budget nachschlagen (auch von rechts nach links!) =XVERWEIS(A2; Kostenstellen!B:B; Kostenstellen!A:A; "nicht gefunden") // Vorteil: Kein #NV mehr durch integrierten Fehlerwert // Vorteil: Sucht in jede Richtung // Vorteil: Gibt mehrere Spalten zurück (Matrix)
Empfehlung: In modernen Umgebungen (Office 365, Excel 2021+) immer XVERWEIS statt SVERWEIS nutzen. Alte Dateien mit SVERWEIS müssen nicht zwingend umgebaut werden — verstehen muss man beide.

03

Pivot-Tabellen — Daten analysieren

Pivot-Tabellen sind das mächtigste Analyse-Werkzeug in Excel. Was früher Stunden mit SUMWENN-Formeln kostete, geht mit Pivot in Minuten — ohne eine einzige Formel schreiben zu müssen.

Pivot in 6 Schritten erstellen

  1. Datenbasis vorbereiten: Tabelle mit Überschriften in jeder Spalte, keine leeren Zeilen/Spalten, jede Zeile = ein Datensatz (z.B. eine Buchung).

  2. Pivot einfügen: Klick in Datentabelle → Einfügen → PivotTable → "Neues Arbeitsblatt" → OK.

  3. Felder anordnen: Im PivotTable-Feldbereich die gewünschten Felder in Zeilen, Spalten, Werte und Filter ziehen.

  4. Wertfeldeinstellung: Rechtsklick auf Wertefeld → "Wertfeldeinstellungen" → Summe, Anzahl, Mittelwert etc. wählen.

  5. Layout anpassen: PivotTable-Design → kompaktes oder tabellarisches Layout wählen. "Teilergebnisse" ein-/ausblenden.

  6. Pivot aktualisieren: Rechtsklick → Aktualisieren (bei Änderung der Quelldaten). Oder: Analysieren → Alle aktualisieren.

Profi-Tipp: Wandle deine Rohdaten zuerst in eine "intelligente Tabelle" um (Strg+T) und benne sie z.B. "tbl_Buchungen". Pivot-Tabellen auf Basis solcher Tabellen aktualisieren sich automatisch, wenn neue Zeilen hinzukommen.

Pivot-Stolperfallen

  • Zahlen werden als Text erkannt: → Wertfeldeinstellung zeigt "Anzahl" statt "Summe" → Rohdaten prüfen, Zahlenformat korrigieren.
  • Datumsgruppierung fehlt: Rechtsklick auf Datum in Pivot → Gruppierung → Jahre/Quartale/Monate auswählen.
  • Leere Zellen in Rohdaten: Entstehen in Pivot als "(leer)" → Daten bereinigen oder Filter setzen.
  • Formatierung geht verloren nach Refresh: → PivotTable-Optionen → "Zellformatierung beim Aktualisieren beibehalten" aktivieren.
04

Bedingte Formatierung — Abweichungen visualisieren

Gute Controller zeigen Abweichungen nicht nur in Zahlen — sie machen sie sofort sichtbar. Bedingte Formatierung ist das einfachste Mittel, um Reports auf einen Blick lesbar zu machen.

Die wichtigsten Regeln im Überblick

Ampel-Farben (3 Regeln)
Grün = Zielerfüllung ≥ 95%, Gelb = 85–95%, Rot = <85%. Ideal für KPI-Übersichten.
Start → Bedingte Formatierung → Neue Regel
Datenbalken
Visualisiert Größenverhältnisse direkt in der Zelle. Kein separates Diagramm nötig. Gut für Kostenstellen-Vergleiche.
Bedingte Formatierung → Datenbalken
Farbskalen
Grün-Gelb-Rot-Verlauf über einen Bereich. Gut für Heatmaps (z.B. Umsatz nach Monat × Region).
Bedingte Formatierung → Farbskalen
Formel-basierte Regeln
Maximale Flexibilität: eigene Formeln entscheiden über Formatierung. Z.B. ganze Zeile einfärben wenn Abweichung > 10%.
=ABS((B2-C2)/C2)>0,1

Plan-Ist-Ampel — Schritt für Schritt

  1. Spalte D enthält die Abweichung in %: =(B2-C2)/C2

  2. Spalte D markieren → Start → Bedingte Formatierung → Neue Regel → "Zellen formatieren, die enthalten"

  3. Regel 1: Zellwert < −0,05 → Rote Füllung (kritisch)

  4. Regel 2: Zellwert zwischen −0,05 und 0,05 → Grüne Füllung (im Plan)

  5. Regel 3: Zellwert > 0,05 → Orange Füllung (Overrun — je nach Kontext gut oder schlecht)

  6. Reihenfolge der Regeln im Manager prüfen — oben stehende Regeln haben Vorrang.

Achtung: Zu viele bedingte Formatierungsregeln verlangsamen Excel erheblich. Bei großen Dateien besser sparsam einsetzen und auf wichtige Spalten beschränken.
05

Einfache Dashboards aufbauen

Ein gutes Controlling-Dashboard fasst die wichtigsten Kennzahlen auf einer Seite zusammen. Es ist kein Kunstwerk — es ist ein Kommunikationsmittel. Klarheit vor Komplexität.

Die 4 Bausteine eines Excel-Dashboards

  • KPI-Übersicht: 3–6 Kennzahlen als große Zahlen mit Trend-Indikator (Pfeile, Ampel)
  • Hauptdiagramm: Säulen- oder Liniendiagramm für den wichtigsten Trend (z.B. Monatsumsatz)
  • Detailtabelle: Plan-Ist-Vergleich der wichtigsten Positionen
  • Filter/Slicer: Interaktive Filter für Zeitraum, Kostenstelle oder Produkt

Praktische Dashboard-Tipps

  • Datenbasis und Dashboard auf getrennten Tabellenblättern halten — das Dashboard zieht nur Werte per Formel.
  • Gridlines ausblenden (Ansicht → Gitternetzlinien deaktivieren) für professionellen Look.
  • Zeilen-/Spaltenköpfe ausblenden auf dem Dashboard-Blatt.
  • Slicer für Pivot-Tabellen verwenden — interaktivere Filterung ohne VBA.
  • Diagramme auf feste Größen snappen: Alt-Taste beim Verschieben/Skalieren.
  • Zellen als Anzeigefelder nutzen: große Schrift, Tausenderpunkte, kein Rahmen → sieht aus wie ein Widget.

A Pivot-Tabellen funktionieren sonst nicht
B Die Pivot-Tabelle aktualisiert sich automatisch, wenn neue Zeilen hinzukommen
C Die Formatierung der Pivot-Tabelle bleibt erhalten
D Excel rechnet dann deutlich schneller
5. Welche Aussage zur bedingten Formatierung ist korrekt?
A Man kann maximal 3 Regeln pro Zellbereich definieren
